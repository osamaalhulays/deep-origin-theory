# Keystone II Arabic Publication Checklist

Use this list before uploading the bundle to a public archive.

## Locked decisions

- Title locked:
  - `نظرية الأصل العميق: A Source-Governed Framework for Curvature Response and Cosmological Dynamics`
- Primary category locked:
  - `gr-qc`
- Theory posture locked:
  - source-governed framework
  - locked kernel
  - auditable response surfaces
- Not to be claimed:
  - theory of everything
  - final observational cosmology
  - unresolved deep-tail completion

## Remaining scientific package tasks

- Confirm final main-paper wording after this editorial decision.
- Keep the scope/open-gap register outside the final public bundle unless explicitly re-approved.
- Decide whether to deposit Markdown only, PDF only, or both.

## Metadata still needed

- Confirm author name or author line.
- Freeze the final abstract wording.
- Freeze the final keyword list.
- Prepare citation metadata.

## Rights and release still needed

- Confirm the public license or release statement.
- Confirm all included files are approved for public release.

## Archive hygiene

- Keep live-tail development material outside the main deposit.
- Keep internal handoff material outside the main deposit.
- Ensure filenames are stable and human-readable.
- Ensure the bundle contains the editorial decision note and short README.

## Nice-to-have

- Add a rendered PDF version of the main paper.
- Add a short release note summarizing the package.
- Add an English-facing deposit summary if the main manuscript remains Arabic.
