# Deep Origin Theory / English Source Layer Index

This file is the English master index for source visibility across the approved English final bundles.

It distinguishes five things clearly:

- authored publication materials inside each bundle
- internal theory-line dependencies
- governed artifact families inherited from the internal archive
- external literature anchors that are actually surfaced in the package
- material that is named but not yet elevated into a source or claim

It also distinguishes a sixth thing:

- outward external resources that are suitable for acknowledgment or related-resource linkage without being promoted into direct source authority
